🎮 Gamepad Raw Input Events – Test Page

How to test

  1. Enable the experimental feature flag in your browser (if required). In Microsoft Edge go to edge://flags/#enable-experimental-web-platform-features and set it to Enabled, then relaunch the browser.
  2. Connect a gamepad to your computer via USB or Bluetooth and make sure the operating system recognises it.
  3. Load or reload this page, then press any button on the gamepad to satisfy the required gamepad user gesture. The status indicator below will turn green once the device is detected.
  4. Move the sticks and press buttons. Each rawgamepadinputchange event will appear in the Event Log below, with coloured tags for changed axes and buttons.
  5. Use the Max log entries slider to limit how many events are kept, and the Clear log button to reset the display.

⚠️ Note: The rawgamepadinputchange event is a proposed API currently being standardised. It may not be available in your browser yet. If the API available indicator below shows Not supported, the page will fall back to the standard polling model so you can still observe gamepad state changes.

⚠️ rawgamepadinputchange is not supported in this browser. The log below is driven by a polling fallback using requestAnimationFrame instead.
