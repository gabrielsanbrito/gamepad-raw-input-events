🎮 Gamepad Raw Input Events – Test Page

How to test

  1. Enable the experimental feature flag in your browser (if required). In Microsoft Edge go to edge://flags/#gamepad-raw-input-change-event and set it to Enabled, then relaunch the browser. In Chrome, go to chrome://flags/#gamepad-raw-input-change-event.
  2. Connect a gamepad to your computer via USB or Bluetooth and make sure the operating system recognises it.
  3. Load or reload this page, then press any button on the gamepad to satisfy the required gamepad user gesture. The status indicator below will turn green once the device is detected.
  4. Move the sticks and press buttons. Each gamepadrawinputchanged event will appear in the Event Log below, with coloured tags for changed axes and buttons.
  5. Use the Max log entries slider to limit how many events are kept, and the Clear log button to reset the display.
